# Dr Magu CLI v1.6.0

## Multi-Agent Orchestrator

v1.6.0 coordinates multiple specialized agents into execution pipelines.

### Added

- MultiAgentPlanner
- MultiAgentOrchestrator
- `multiagent.plan`
- `multiagent.run`
- `dr-magu multiagent-plan`
- `dr-magu multiagent-run`
- SDLC pipeline
- Research-to-build pipeline

### Goal

Move Dr Magu from single-command execution toward coordinated multi-agent workflows.
