# Dr Magu CLI v1.1.2

v1.1.2 enables default-model chat execution.

- LLMRuntime
- OpenAICompatibleProvider
- `llm.chat`
- `dr-magu llm-chat`

The runtime is OpenAI-compatible and uses the existing default model configuration. Research remains deterministic until the MCP Research Runtime version.
